# neutrino_emojis_other

Original generated CC0-style emoji/category icon pack for the Neutrino Project.

This delivery is split into one core archive and one PNG archive per theme so the package remains downloadable and practical. Extract the core archive, then extract any or all theme archives into the same destination; the folders merge under `neutrino_emojis_other/`.

## Provenance

The artwork is original deterministic geometry generated by `tools/generate_neutrino_emojis_other.py`. No third-party emoji/icon artwork is bundled.

Country flag icons are symbolic country-code flag tiles with deterministic stripe geometry. They are normalized UI assets, not official government flag artwork. Military and weapon-related entries are simplified, non-operational emoji symbols only.
